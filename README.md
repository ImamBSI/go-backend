Repositori ini berisi layanan backend yang dikembangkan menggunakan Go (Golang) dan framework Fiber. Proyek ini dirancang dengan fokus pada performa tinggi, skalabilitas, dan struktur kode yang modular.

🚀 Fitur Utama
High Performance: Dibangun di atas Fasthttp (melalui Fiber).

Modular Architecture: Pemisahan yang jelas antara handlers, models, dan database logic.

Authentication: Integrasi JWT (JSON Web Token) untuk pengamanan endpoint.

Database ORM: Menggunakan GORM untuk interaksi database yang efisien.

Environment Configuration: Manajemen konfigurasi menggunakan file .env.
